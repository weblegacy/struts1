Application Tag Library

Standard Syntax:
     <%@ taglib prefix="app" uri="http://jakarta.apache.org/taglibs/struts-example-1.0" %>

XML Syntax:
     <anyxmlelement xmlns:app="http://jakarta.apache.org/taglibs/struts-example-1.0" />
This tag library contains functionality specific to the Struts Example Application.

Tag Library Information
Display NameApplication Tag Library
Version1.0
Short Nameapp
URIhttp://jakarta.apache.org/taglibs/struts-example-1.0
 

Tag Summary
checkLogon Validate that there is a currently logged on user, by checking for the existence of a session-scope bean under the specified name. If there is no such bean, forward control to the specified page, which will typically be a logon form.
linkSubscription Generate a URL-encoded hyperlink to the a URL made up of the specified base value, plus optional query parameters that select the specified Subscription object.
linkUser Generate a URL-encoded hyperlink to the a URL made up of the specified base value, plus optional query parameters that select the specified User object.
